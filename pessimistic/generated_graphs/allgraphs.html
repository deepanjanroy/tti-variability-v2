- The graphs are sorted by increasing mean TTI

- For each graph, TTI values are sorted in increasing order.

- Why doesn't some sites have 25 runs?

We did not get a TTI value for all 25 runs. There could be multiple reasons for this - we did not wait long enough for the site to reach firstInteractive, there was some network/device error and the site did not load at all, the site somehow spawned multiple renderers, both of which reached TTI and my code did not deal with multiple TTI values, etc. It's mostly because we didn't wait long enough.

- Why is TTI plotted on the x axis?

All the vertically stacked graphs share a common x-axis, and it's easier to compare TTI distribution of different sites this way. We could achieve the same thing by plotting TTI on the y axis and stacking the graphs horizontally, but horizontal scrolling is awkward.
